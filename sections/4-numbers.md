## 4. Numbers

```elm
42
-- 42 : number

3.14
-- 3.14 : Float
```

In Elm, a `number` is either an `Int` or a `Float`.

```elm
2 + 5
-- 7 : number

3 * 2
-- 6 : number

2 + 5 - 3 * 2
-- 1 : number

(2 + 5 - 3) * 2
-- 8 : number

9 / 2
-- 4.5 : Float

9 % 2
-- 1 : Int

9 // 2
-- 4 : Int

4^2
-- 16 : number

Basics.round 3.14
-- 3 : Int
```
